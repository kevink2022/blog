Build Simple, Composable Systems

“Familiarity hides complexity.”

— Rich Hickey1

“Civilization advances by extending the number of important operations which we can perform without thinking about them.”

— Alfred North Whitehead2

“Any fool can write code that a computer can understand. Good programmers write code that humans can understand.”

— Martin Fowler3

“Newlines are cheap; people’s time is expensive.”

— Brooklyn Data Co. SQL Style Guide4

“The price of reliability is the pursuit of the utmost simplicity.”

— C. A. R. Hoare5

Optimize for Fast Feedback Loops

“Everything designed by humans will have flaws. So design your systems so the flaws are quickly evident. When they do, respect that as a bonafide signal of opportunity to get better.”

  • Steven Spear, Idealcast 5 (24:13) 1
